## Cleaning Data

<Steps>
<Step>
### Format Column `timestamp`

<Callout type="info">
Menggunakan format tanggal [ISO 8601](https://en.wikipedia.org/wiki/ISO_8601): `YYYY-MM-DD hh:mm:ss`
</Callout>

1. Block column B
2. Klik `More Format` (icon `123`)
3. Pilih date time format `YYYY-MM-DD hh:mm:ss` atau `2025-02-04 07:37:16`
4. Selesai
</Step>

<Step>
### Rapihkan Column `phone`

Pastikan nomor telepon dimulai dengan `628` bukan `8`.

Gunakan **Rubot → Data Processing → Cleaning Phone Numbers**
</Step>

<Step>
### Rapihkan Column `name`

Gunakan **Rubot → Data Processing → Cleaning Text**
</Step>

<Step>
### Penyesuaian `source`, `sremarks`, `medium`, `mremarks`

<Callout type="error">
Jika event belum dibuat di Event Manager [budget.kyzn.life](https://budget.kyzn.life/dl/event-manager), input event tersebut dulu untuk mendapatkan `sremarks` dari `event_id`
</Callout>

**Cara Mengcopy `event_id`:**
1. Buka https://budget.kyzn.life/dl/event-manager
2. Klik Event detail terkait
3. Klik `Copy`

### Hapus Duplicate

1. Klik menu `Data`
2. Pilih `Data clean-up`
3. Pilih `Remove Duplicate`
4. Hanya pilih `column E` (column `phone`)
5. Klik `Remove Duplicate`
</Step>
