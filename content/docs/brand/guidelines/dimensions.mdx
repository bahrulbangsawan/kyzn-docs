## Best Practices

- Selalu export dalam resolusi tinggi (2x untuk retina)
- Gunakan format yang sesuai: PNG untuk transparansi, JPG untuk foto
- Compress gambar untuk web tanpa mengorbankan kualitas
- Simpan file master dalam format editable (PSD, AI, Figma)
